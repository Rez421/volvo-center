# Volvo Center

Профильный сервисный сайт для автомобилей Volvo.

Проект содержит лендинг с описанием сервиса, услуг, отзывов, FAQ, формой заявки, каруселью моделей Volvo, блоком «О нас», разделом с этапами работы, секцией контактов и картой.

## О проекте

Volvo Center — профильный автосервис для владельцев Volvo, которым важны аккуратное обслуживание, понятные рекомендации и честная диагностика.

Сайт создан для того, чтобы показать услуги центра, подчеркнуть профильную специализацию по Volvo и дать пользователю удобный способ оставить заявку.

Сервис ориентирован только на автомобили Volvo, представленные на сайте. В проекте есть подробное описание услуг, этапов работ, частые вопросы, отзывы клиентов и форма записи на обслуживание.

## Возможности

- Адаптивная вёрстка под desktop, tablet и mobile.
- Hero-блок с разделением на текстовую часть и блок моделей Volvo.
- Карусель моделей Volvo.
- Блок «О нас» со статистикой компании.
- Раздел «Услуги и цены» с аккордеоном услуг.
- Раздел «Как работаем» с пошаговым процессом обслуживания.
- Отзывы клиентов.
- FAQ с частыми вопросами.
- Форма заявки на диагностику или ремонт.
- Блок контактов и карта.
- Favicon для сайта.
