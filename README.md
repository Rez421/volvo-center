# Volvo Center

Профильный сервисный сайт для автомобилей Volvo.  
Проект содержит лендинг с описанием сервиса, услуг, отзывов, FAQ, формой заявки и каруселью моделей Volvo.

## О проекте

Сайт сделан для автосервиса Volvo Center.  
Основная задача — показать услуги центра, подчеркнуть профильную специализацию по Volvo и дать пользователю удобный способ оставить заявку.

## Возможности

- Адаптивная вёрстка под desktop, tablet и mobile.
- Hero-блок с разделением на текстовую часть и блок моделей.
- Карусель моделей Volvo.
- Блоки с услугами, этапами работы, отзывами и FAQ.
- Форма заявки на обслуживание.
- Карта и контактная информация.

## Запуск

1. Скачай или открой проект в редакторе.
2. Открой файл `index.html` в браузере.
3. Либо запусти проект через Live Server в VS Code или PyCharm.
